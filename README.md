# NC Numerology App

A Flutter-based numerology application that calculates personal numbers using name and birthdate with interactive UI elements.

---

## 🚀 Features
- Name-based number calculation
- Birthdate life path analysis
- Numerology number reduction system (1–9, 11, 22, 33)
- Interactive UI design (wheel system planned)

---

## 🛠 Tech Stack
- Flutter (Dart)
- State Management (to be added)
- Clean Architecture (feature-based structure)
